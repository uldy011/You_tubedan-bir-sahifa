# YouTube Clone

Bu loyiha YouTube saytining bosh sahifasini HTML va CSS yordamida qayta yaratilgan (klon qilingan) versiyasidir.

## Sayt haqida

Ushbu sayt YouTube platformasining asosiy ko'rinishini taqlid qilgan holda yaratilgan bo'lib, quyidagi qismlardan iborat:

- **Yuqori panel (Header)** — YouTube logotipi va foydalanuvchi profili
- **Yon menyu (Sidebar)** — Home, Shorts, Subscriptions, History, Playlists, Your videos, Courses kabi bo'limlar
- **Kategoriyalar paneli** — Trending, Music, Gaming, Programming, Live, Mixes, News, Podcasts kabi filtrlar
- **Video kartalar (Video Cards)** — video rasmi (thumbnail), kanal avatari, video nomi, kanal nomi, ko'rishlar soni va yuklangan vaqti ko'rsatilgan kartalar

## Ishlatilgan texnologiyalar

- **HTML** — sahifaning umumiy strukturasini yaratish uchun
- **CSS** — sahifani uslublash, video kartalarni va menyularni joylashtirish (layout) uchun

## Loyihaning maqsadi

Ushbu loyiha real dunyoda mavjud bo'lgan murakkab saytlarning tuzilishini o'rganish va HTML/CSS asosida responsiv (moslashuvchan) dizayn yaratish ko'nikmalarini mustahkamlash maqsadida bajarilgan.
